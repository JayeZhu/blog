vue-cli解决跨域

# proxy

vue-cli 可以使用代理实现跨域，从以前的 Nginx 转变过来，设置方便

# 设置

module.exports = {
  devServer: {
    proxy: {
      '/api': {
        target: '/http://10.xxx.x.xx:3000', // 代理接口地址
        pathRewrite: { '^/api: '' },
      }
    }
  }
}
