数组遍历

# 前言

开发中，经常需要对数组每一项进行操作，针对不同场景，用适合的方法更好

let arr = [1, 2, 3];
1

# for

for (let i = 0, len = arr.length; i < len; i++) {
  console.log(`${i}: ${arr[i]});
}
// 0: 1
// 1: 2
// 2: 3

# Array.ptototype.forEach()

arr.forEach((i, index) => {
console.log(`${index}: ${i});
});
1

遍历数组每一项，不返回值，对原数组没有影响。适合操作数组不生成值也不影响原数组的场景，是很友好的方法。

# Array.ptototype.map()

let arrMap = arr.map(i => i * 2); // -> [2, 3, 6]
