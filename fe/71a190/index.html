CSS居中

提示

让元素居中，是布局中常见的需求，需要掌握不同情况下的居中方法。

行内元素 html：

<div class="parent">
  <span class="child">👋 Hello World!</span>
</div>
1
2
3

块级元素 html：

<div class="parent">
  <div class="child">👋 Hello World!</div>
</div>
1
2
3

# 水平居中

# text-align

对于 display 为 inline、inline-block、inline-table、inline-flex 等行内元素， 可以直接对其父元素使用 text-align 属性实现居中。

.parent {
  text-align: center;
}
1
2
3

限制：行内元素
传送门 (opens new window)

# margin

在宽度设定的块级元素的样式中，将左右外边距设置为 auto 可以实现水平居中

.child {
  width: 200px;
  margin: 0 auto;
}
1
2
3
4

限制：宽度设定,块级元素
传送门 (opens new window)

# 子元素含 float

子元素存在左浮动，为了让子元素居中，可以让父元素宽度设置 fit-content，并且配合 margin，可以实现居中

.parent {
  width: -moz-fit-content;
  width: -webkit-fit-content;
  width: fit-content;
  margin: 0 auto;
}
.child {
  float: left;
}

限制：fit-content 存在兼容性问题

# flex

弹性盒实现水平居中是极其容易的，只需设置父元素即可

.child {
  display: flex;
  justify-content: center;
}

# transform

利用 CSS3 中新增的转换属性 transform 实现居中

  position: relative;
}
.child {

# 总结

  • 行内元素居中简单，设定 text-align 和 line-height 即可
  • 宽高固定的块级元素可以使用 margin 实现居中
  • 宽高不固定可以使用 transform 实现居中
  • 不考虑兼容性，flex 布局实现居中最为简便
  • fit-content 和 vertical-align 比较少用的，但限制较少，较为理想
