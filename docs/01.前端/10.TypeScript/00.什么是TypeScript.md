## TypeScript 简介
前端开发直接 JavaScript 一把梭哈，为什么微软又开发出 TypeScript 呢，是为了卷吗。不不不，它是给 js 添加特性的，是 js 的超集，支持 ES6 标准（好耶）。下面在仔细看看这家伙

## typescript 特性
ts 对 js 语言进行拓展了，包括：
- 类型批注和编译时类型检查
- 类型判断
- 类型擦除
- 接口
- 枚举
- Mixin
- 泛型编程
- 名字空间
- 元祖
- Await
