变量对象（Variable Object，VO）是执行上下文重要组成部分——数据作用域，
存储了在执行上下文中定义的所有变量声明和函数声明，保证代码执行时对变量和
函数的正确访问。

## 变量对象种类
通过不同执行上下文，可以简单分为：
- 全局执行上下文对应的全局执行上下文变量对象
- 函数执行上下文对应的函数执行上下文变量对象

### 全局执行上下文变量对象
其中全局执行上下文变量对象其实就是全局对象，在浏览器中指的是 window 对象，
